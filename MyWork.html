Personal Website

Hosted on GitHub sites, this personal site is built as light as possible. Foundation handles most of the boilerplate css, while I took care of most of the page specific css in order to keep the UX and UI as clean as possible.

Foundation handles verly little of the javascript which I custom tailored for each page (including the parallax scrolling seen here). Since there is no backend, the contact page is currently handled by formspring.io. The eventual goal is to have enough functionality to warrant a backend, built on either ruby or python.
